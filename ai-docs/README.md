# AI Operating System

Este diretório centraliza as regras que todas as IAs, agentes e colaboradores devem seguir ao trabalhar no Flux Agent Studio.

## Objetivo

Criar uma memória organizacional compartilhada.

Antes de aumentar a quantidade de modelos (Claude, GPT, Gemini, etc.), garantimos que todos operem com os mesmos princípios, regras de negócio, padrões de UX e arquitetura.

- ux-rules.md

- roadmap-ai-orchestration.md

## Filosofia

O gargalo atual não é inteligência.

O gargalo é consistência.

Primeiro padronizamos.
Depois escalamos.
Depois orquestramos múltiplos modelos.
