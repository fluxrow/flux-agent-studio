Hero Loop Recorder — Flux Agent Studio

Pronto para gravar.
Como usar:

1. Clique em ⏺ Gravar WEBM — a animação começa e é gravada por 10 segundos.
2. Aguarde a gravação terminar (ou clique em ⏹ Parar antes).
3. Um link de download hero-loop.webm aparecerá abaixo.
4. Clique em 📸 Exportar Poster para baixar hero-loop-poster.jpg.
5. Para MP4: abra o WEBM no HandBrake (gratuito) ou use ffmpeg -i hero-loop.webm hero-loop.mp4.
6. Copie os 3 arquivos para public/ do projeto.

Alternativa MP4 online: cloudconvert.com/webm-to-mp4 (gratuito, sem conta)
